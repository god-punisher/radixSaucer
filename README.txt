O programa recebe como entrada dois datasets (entradas/frankestein_clean.txt e entradas/war_and_peace_clean.txt).

O programa gera como saída dois arquivos de texto (saidas/frankenstein_ordenado.txt e saidas/war_and peace_ordenado.txt) listando em ordem lexicografica as palavras contendo apenas letras do alfabeto ingles, isto é, numerais e símbolos são ignorados. Além disso, é gerado um segundo arquivo para cada dataset, (ocorrencias/frankenstein_ordenado.txt e ocorrencias/war_and_peace_ordenado.txt), que conta o numero de ocorrências de cada palavra apenas varrendo o vetor gerado pelo algoritmo de ordenação.

Obs: Antes de executar, excluir os arquivos dentro das pastas "ocorrencias" e "saidas", mas manter as pastas!
